vividkps
vividkps is a very bad implementation of a keys per second (KPS) display in python for rhythm games, drawing strong inspiration from vivid/stasis for it's name and original image setup

this project is meant to operate similarly to the now un-supported osukps project (which still does work and can be found at the following repository link: https://github.com/yugecin/osukps)

note that this project was my first proper foray into programming a tkinter thing, and thus is likely subpar quality. this project will hopefully be revisited in the future.
vividkps requires the following python libraries to operate:
  • PILLOW
  • keyboard
